# Smart flower pot Monitoring System using ESP32

This project is an ESP32-based IoT system that performs real-time monitoring using sensors and cloud communication. It includes touch and vibration detection and audio recording capabilities.

## Purpose
To develop a hidden IoT-based monitoring system for real-time safety alerts in home environments.

## Features
* Touch sensor-based detection
* Vibration detection system
* Real-time alerts using Blynk IoT platform
* Audio recording using digital microphone
* Data storage using SD card

## Components Used
- ESP32 DevKit V1
- NodeMCU ESP8266
- Vibration sensor module
- Touch sensor module
- INMP441 digital microphone
- MicroSD card module

## Technologies Used
- Arduino IDE
- Blynk IoT platform
- Embedded C and C++ (Arduino framework)

## Communication Protocols
- SPI  – SD card module communication
- I2S – audio data interface
- WiFi – cloud communication
